# pugilator_transcriptomics
This repository contains the analysis of our Leptuca pugilator transcriptomics work from the summers of 2021 and 2024. This contains the analysis of larval gene expression under different light conditions. 

## Study Structure & Sample Collection
Ovigerous _Leptuca pugilator_ were collected from Oyster Landing at the Belle Baruch Marine Field Station in Georgetown, SC.
